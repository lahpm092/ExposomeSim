A world event is felt twice — once instantly by the body (low road), once interpreted by the LLM, which appraises it (SEC/OCC) and chooses how to cope. Both routes inject impulses into the soma: a persistent set of limbic, neuromodulator, hormone and drive variables that keep evolving as coupled stochastic equations. Core affect is read off the soma, the discrete emotion is constructed from it, the integrals accumulate the lived dose, and the felt state is fed back as the next prompt. Beneath the soma a homeostatic reservoir layer generates felt hunger, thirst and urgency; Maslow needs read off the substrate drive a utility arbiter, so the daily loop emerges; and memory is an affective graph that consolidates episodes into gists and schemas.
